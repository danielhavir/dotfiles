# dotfiles

Run:

```shell script
git clone https://github.com/danielhavir/dotfiles $HOME
cd $HOME/dotfiles/
bash setup.sh
cd $HOME
rm -rf dotfiles/
```

Or alternatively single line setup:
```shell script
curl -fsSl https://danielhavir.github.io/setup.sh | sh
```
